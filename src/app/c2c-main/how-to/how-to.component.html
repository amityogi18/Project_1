C2C Common Library How-To Guide

Creating a Common Component

In Common Project:

  1. Create a new Module with the CLI:
      ng g m module-name
  2. Create a new Component with the CLI (same directory as module):
      ng g c component-name
  3. Add component-name to the exports array in the new module.
  4. Add an export for module-name to the public_api.ts file.
  5. Add component-name to the Declarations array in app.module.ts
  6. Add a new path to component-name in app-routing.module.ts
  7. Run the command to publish file(s) to Nexus (See C2C Version Rules to determine type):
      npm run pub

Importing and Using a Common Component/Directive

  1. Run the command to pull new file(s) from Nexus:
      npm run inst
  2. In app.module.ts, add the module-name to the imports array.
  3. Add the necessary markup to the HTML.
  4. You may need to restart VS Code for it to pick up the changes.
